## Como substituir no GitHub
1. Descarregue este ZIP.
2. Extraia a pasta.
3. No repositório GitHub do site, use **Add file → Upload files**.
4. Arraste todos os ficheiros desta pasta para o repositório.
5. Se o GitHub perguntar, confirme que quer substituir os ficheiros existentes.
6. Clique em **Commit changes**.
