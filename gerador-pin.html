Gerador de PIN aleatório

Gere códigos PIN simples de 4 ou 6 dígitos para exemplos, testes, protótipos e simulações. Não substitui sistemas de segurança reais.

Escolha o formato e clique em gerar.
Pode usar esta ferramenta quantas vezes quiser no telemóvel ou no computador.

Como usar

  • Escolha 4 ou 6 dígitos.
  • Indique quantos códigos quer gerar.
  • Clique no botão para ver os PIN.

FAQ rápida

Os códigos são únicos?

Nesta versão tenta gerar uma lista sem repetir dentro do mesmo lote.

Posso usar estes PIN para segurança real?

Não é aconselhável. É uma ferramenta simples para exemplos e testes.
