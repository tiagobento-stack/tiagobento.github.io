Privacidade

Este site foi concebido para ser simples. As ferramentas funcionam diretamente no navegador e, por regra, os dados que introduz não precisam de ser enviados para um servidor externo.

Podem existir dados técnicos básicos associados ao funcionamento normal do alojamento, como registos de acesso e métricas gerais do serviço. Esses elementos dependem da plataforma onde o site está publicado.

No futuro, este site pode usar ferramentas de estatística ou publicidade. Se isso acontecer, esta página deverá ser atualizada para refletir essas mudanças.

Ao continuar a usar o site, assume que compreende que qualquer informação colocada nas ferramentas serve apenas para gerar resultados na própria página.
