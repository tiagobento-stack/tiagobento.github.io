Regra de três simples

A regra de três simples ajuda a encontrar um valor desconhecido quando duas grandezas mantêm proporção. É útil para preços, quantidades, receitas e conversões rápidas.

Preencha A, B e C para calcular X.
Pode usar esta ferramenta quantas vezes quiser no telemóvel ou no computador.

Como usar

  • Preencha os três valores conhecidos.
  • Pense na relação A : B = C : X.
  • Clique em calcular para obter X.

Ferramentas relacionadas

  • Calculadora de percentagens
  • Calculadora de média de notas
  • Simulador de poupança
  • Planeador de orçamento mensal
